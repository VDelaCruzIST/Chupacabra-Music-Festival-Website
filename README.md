# Chupacabra Music Festival Website

## Overview
This repository contains a music festival promotional website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website advertises a fictional music festival and demonstrates advanced front-end web development concepts, including CSS 3D transformations, perspective effects, layered layouts, navigation systems, and interactive visual design.

## Features
* Music festival promotional website
* CSS 3D cube effect
* Perspective and transformation styling
* Multi-column page layout
* Social media integration
* Event information panels
* Custom branding and graphics
* Interactive visual design

## Technologies Used
* HTML5
* CSS3
* CSS 3D Transforms
* CSS Perspective Effects

## Project Files
* `cf_home.html` – Main festival webpage
* `cf_reset.css` – CSS reset stylesheet
* `cf_layout.css` – Layout and page structure
* `cf_effects.css` – Visual effects and 3D styling
* Festival branding and image assets

## Skills Demonstrated
* Front-end web development
* CSS 3D transformations
* Perspective and visual effects
* User interface design
* Multi-column layouts
* Navigation design
* Website content organization
* Semantic HTML development
